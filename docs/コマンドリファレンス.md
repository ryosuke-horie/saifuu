# コマンドリファレンス

## 概要

Saifuuプロジェクトで使用する全コマンドの包括的なリファレンスです。

## プロジェクトセットアップ

### 初回セットアップ

```bash
# Node.js 22のインストール（mise使用）
mise install

# 依存関係のインストール（ルート）
npm install

# API依存関係のインストール
cd api && npm install

# フロントエンド依存関係のインストール
cd frontend && npm install

# 環境変数ファイルの作成
cp api/.env.example api/.env
cp frontend/.env.example frontend/.env

# 開発環境データベースのセットアップ（初回のみ）
cd api && npm run db:setup:dev
```

## 開発サーバー

### 通常の開発

```bash
# フロントエンド開発サーバー（ポート3000）
npm run dev

# API開発サーバー（ポート5173）
cd api && npm run dev

# Node.js版APIサーバー（デバッグ用）
cd api && npm run dev:node
```

### E2E開発環境

```bash
# E2E用フロントエンドサーバー（ポート3002）
cd frontend && npm run dev:e2e

# E2E用APIサーバー（ポート3003）
cd api && npm run dev:e2e
```

### バックグラウンド実行（Ghost使用）

```bash
# 開発サーバーをバックグラウンドで起動
ghost run npm run dev

# Storybookをバックグラウンドで起動
ghost run npm run storybook

# プロセス管理
ghost list                     # 実行中のプロセス一覧
ghost log <task-id>           # ログ確認
ghost stop <task-id>          # プロセス停止
ghost cleanup --days 7        # 古いタスクのクリーンアップ
```

## 環境変数

### API環境変数（api/.env）

```bash
# 開発環境用SQLiteデータベースパス
DEV_DB_PATH=./dev.db

# 環境フラグ
NODE_ENV=development
```

### フロントエンド環境変数（frontend/.env）

```bash
# API Base URL
NEXT_PUBLIC_API_URL=http://localhost:5173/api

# API設定
NEXT_PUBLIC_API_TIMEOUT=30000
NEXT_PUBLIC_API_MAX_RETRIES=3
NEXT_PUBLIC_API_RETRY_DELAY=1000
NEXT_PUBLIC_API_PORT=5173
```

## トラブルシューティング

### ポート使用中エラー

```bash
# 使用中のポートを確認
lsof -i :3000
lsof -i :5173

# プロセスを終了
kill -9 <PID>
```

### データベースエラー

```bash
# 開発DBリセット
cd api && npm run db:reset:dev

# E2E DBリセット
cd api && rm e2e-test.db
```

### 依存関係エラー

```bash
# クリーンインストール
rm -rf node_modules package-lock.json
npm install
```

## 注意事項

- **Ghost使用推奨**: 長時間実行プロセスは必ずGhostでバックグラウンド実行
- **環境分離**: 開発・E2E・本番環境は明確に分離
- **型安全性**: 必ず型チェックを通してからコミット
- **テスト実行**: PR作成前に全テストの実行を推奨
